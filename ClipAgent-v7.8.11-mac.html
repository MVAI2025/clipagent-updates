ClipAgent release notes

Version 7.8.11

  • Fixed Twitch creator live downloads that were failing because the packaged Streamlink binary was missing the lxml dependency.
  • Moved Twitch creator/watcher recording back to the more reliable yt-dlp path by default, including automatic migration for older saved creators that were still pinned to Streamlink.
  • Kept the editor-safe YouTube path intact so downloads still prefer H.264/AAC MP4 and skip re-encode whenever the source is already NLE-safe.
