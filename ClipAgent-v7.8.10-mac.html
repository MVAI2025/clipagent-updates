ClipAgent release notes

Version 7.8.10

  • Fixed the packaged backend crash on launch by bundling the app version file into the PyInstaller backend and adding a safe runtime fallback.
  • Stopped the startup splash from hanging after a backend crash and now surface the real backend error plus the local backend log path.
  • Aligned the desktop default port to 8787 and automatically avoid the old 8000 conflict when migrating legacy settings.
