ClipAgent release notes

Version 7.8.13

  • Fixed Twitch live recordings that could be saved as a transport stream with an .mp4 extension, which made the file appear corrupted or unwatchable.
  • ClipAgent now verifies the final container and remuxes live recordings into a real playable MP4 before delivery.
  • Updated creator status copy so it is clear that the on-disk file is finalized after you stop the recording.
